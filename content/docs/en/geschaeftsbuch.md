# Business log

The business log documents all official transactions and records of the fire department.

![Business log](/docs-assets/screenshots/geschaeftsbuch.png)

## Features

- **Document incoming and outgoing transactions** In/Out toggle
- Automatic sequential numbering
- **From/To fields** Record sender and recipient
- Forwarding to staff functions (S1–S6)
- **Tabs per staff function** All entries, S1 personnel, S2 situation, S3 operation, S4 logistics, S5 public relations, S6 communication
- Read marker per staff function
- Done timestamp
- Sort by every column
- CSV export

## Instructions

### 1. Create an entry

1. The number is assigned automatically
2. Choose In/Out (incoming or outgoing)
3. Fill in From/To
4. Enter the name = subject/information
5. Optionally add a description
6. Enter forwarding/marking (e.g. "S1, S3")
7. Click Add

:::info
Tip: Use "In" for incoming messages (e.g. situation reports) and "Out" for outgoing commands or information.
:::

### 2. Filter by staff function

1. Select the tabs at the top: S1–S6 only show relevant entries based on the forwarding field

:::info
Tip: The S functions (S1–S6) correspond to the staff functions in the command staff. Entries with "S1, S3" in the forwarding field appear in both tabs.
:::

### 3. Mark as read

1. Click the check icon next to an entry to mark it as read for your own function

### 4. Mark an entry as done

1. Set the done timestamp

### 5. Sort

1. Click a column heading: number, date, From/To, name, description, done

### 6. Export as CSV

1. Click the download button

### 7. Edit or delete an entry

1. Click the entry in the table to edit it
2. Make changes and save, or delete the entry
