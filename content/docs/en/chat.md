# Chat

The chat enables real-time communication between all responders during an operation. Messages are tied to the active operation and visible to everyone involved.

## Features

- Send and receive real-time messages
- Push notifications for new messages (via Firebase Cloud Messaging)
- Toggle chat notifications on/off
- Messages with sender and timestamp
- Automatic assignment to the active operation

## Guide

### Open the chat

1. Click "Chat" in the menu
2. The chat shows messages for the active operation

### Send a message

1. Type text into the input field
2. Press Enter or click the send button
3. The message appears immediately for all participants

### Enable notifications

1. Activate the "Chat notifications" switch at the top of the chat
2. You will receive a push notification for new messages

### Disable notifications

1. Turn the switch off if you no longer want to be notified

:::info
Tip: Chat messages are always tied to the active operation. Switch the operation to see the chat of another operation.
:::

:::warning
Note: Push notifications require the app to have notification permission. Allow notifications in the browser or on your mobile device.
:::
