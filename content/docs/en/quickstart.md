# Quick start

This guide walks you through creating a new operation and using the most important features in just a few steps.

## 1. Create a new operation

Click the fire icon in the top navigation bar to create a new operation.

![Fire icon in the navigation bar](/docs-assets/screenshots/quickstart-firecall-button.png)

A dialog opens where you can enter the operation data. Fill in the fields (address, operation type, etc.) and save the operation.

![Dialog to create a new operation](/docs-assets/screenshots/quickstart-firecall-dialog.png)

## 2. Enable editing mode

To add elements to the map, enable editing mode by clicking the pencil icon in the map view.

![Enable editing mode](/docs-assets/screenshots/quickstart-edit-mode.png)

## 3. Add a vehicle

In editing mode you can add vehicles to the map. Select the vehicle type (e.g. "TLFA") and the fire department (e.g. "Neusiedl am See").

![Add a vehicle](/docs-assets/screenshots/quickstart-add-vehicle.png)

## 4. Create a diary entry

Open the sidebar and create a new diary entry to document important events during the operation.

![Diary entry in the sidebar](/docs-assets/screenshots/quickstart-diary-entry.png)

## Next steps

You now know the basics. Explore the other features in the remaining documentation pages or just try the app yourself!
