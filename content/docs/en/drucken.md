# Print

The print function generates a comprehensive operation report that summarises all relevant data of the active operation in a print-optimised layout.

## Features

- Operation header: name, date, fire department, description
- Operation map as an embedded map
- Resource summary: first alert, first arrival, last departure
- Resource details: type, name, details, position (grouped by layer)
- Operation sites with status, addresses, times and vehicles
- Measurements (spectrum/radiation data) if available
- Operation log entries
- Business log entries
- Print-optimised CSS layout with page breaks

## Guide

### Open the print view

1. Click "Print" in the menu
2. The page shows the full operation report

### Review the report

1. Go through all sections: header, map, resources, sites, measurements, log, business log

### Show/hide measurements

1. Use checkboxes to include or exclude individual measurements from the print
2. Checkboxes are only visible on screen

### Print

1. Use the browser print function: Ctrl+P / Cmd+P
2. The print preview shows the optimised layout
3. Save as PDF or print directly

:::info
Tip: The print view is optimised for A4 portrait. Use "Save as PDF" in the print dialog for a digital copy.
:::

:::info
Tip: Sections without data (e.g. an empty log) are hidden automatically.
:::
