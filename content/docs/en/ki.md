# AI assistant

The AI assistant uses artificial intelligence to answer questions about the current operation, create summaries and help with public relations work. The assistant has access to all data of the active operation.

## Features

- Ask questions about the operation and get AI-generated answers
- Automatically create operation summaries
- Generate social media posts for Facebook / Instagram
- Different assistant modes (default, social media)
- Streaming answers in real time
- **Access to operation data** Name, date, description, vehicles, personnel, operation log, business log

## Instructions

### Open the AI assistant

1. Click "AI" in the menu
2. The page shows an input field and the assistant selection

### Choose an assistant mode

1. Use the "Assistant" dropdown
2. **Default** for general questions
3. **Social media** for Facebook / Instagram posts

### Ask a question

1. Enter a question or task in the text field, e.g. "Create a summary of the operation"
2. Press Enter or click the "Ask" button
3. The answer streams in real time

### Create social media posts

1. Switch the mode to "Social media"
2. Enter, for example, "Create a Facebook post about the operation"
3. The AI generates a post with matching hashtags and tone

### Use the results

1. The answer is displayed as formatted text
2. The text can be copied and reused

:::info
Tip: The AI assistant automatically has access to all data of the active operation – you do not have to enter any details manually.
:::

:::info
Tip: In social media mode the AI creates posts with the right tone, hashtags and target audience for fire department public relations.
:::

:::warning
Note: An active operation must be selected for the AI assistant to use operation data.
:::
