# Hazmat

The hazmat database contains information about dangerous substances and how to handle them in an operation.

![Hazmat database](/docs-assets/screenshots/schadstoff.png)

## Features

- Search hazardous substances by UN number
- Search hazardous substances by substance name
- Open safety data sheets (ERICards) via external link to ericards.net
- **View protective suit parameters** Resistance level, time resistance, damage
- Radiation protection calculator (own tab)
- **Energy spectrum analysis (own tab)** Gamma spectroscopy

## Instructions

### 1. Search a hazardous substance

1. Open the hazmat database tab
2. Enter a UN number OR a substance name
3. Click the search button
4. Review the result list

:::info
Tip: You can search by UN number (e.g. 1203 for petrol) or by substance name.
:::

### 2. Read the results

1. Every card shows: UN number, name, protective-suit parameters with resistance level, time resistance and damage

### 3. Open ERICards

1. Click the "Ericards" button on a result card
2. An external window opens with the complete safety data sheet

:::info
Tip: ERICards (Emergency Response Intervention Cards) contain detailed operation guidance for first responders.
:::

### 4. Use radiation protection

1. Select the "Radiation protection" tab
2. Compute the inverse square law, shielding factor, stay time, dose rate from nuclide activity or unit conversion

:::info
The complete instructions with formulas, examples and notes on the reference values can be found in [Radiation protection](/docs/strahlenschutz).
:::

### 5. Analyse an energy spectrum

1. Select the "Energy spectrum" tab
2. Capture gamma spectroscopy data and identify nuclides

:::info
A detailed guide for peak detection, nuclide identification and chart usage can be found in [Energy spectrum](/docs/energiespektrum).
:::
